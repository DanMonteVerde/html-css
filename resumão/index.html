Elemento picture em HTML5

Exemplo de responsividade com imagens usando o elemento picture serve para que para cada tamanho de tela tenha uma imagen ou o tamanho da mesma so que para uma tela maior ou visse versa. Exemplo abaixo:

        (picture>
            (source media="(max-width: 500px)" srcset="favicon.ico.png">
            (source media="(max-width: 749px)" srcset="ryder.jpeg">
            (img src="testeimg.jpg" alt="">
         (/picture>

Para testar podemos ajustar a tela!

Áudio em HTML5

Para colocarmos um audio no HTML5 colocamos no src o audio que esta na nossa pasta e depois um atributo chamado controls ele ira fazer todos os controles para que o usuario possa reproduzir a musica! EXEMPLO ABAIXO:

        audio src="Ed_Sheeran_-_Perfect_(BornMP3.com).mp3" controls>/audio

Videos em HTL5 em hospedagem propria.

Para colocarmos um video com hospedagem propria é exatamente igual a colocar um audio a diferença é que usamos a tag VIDEO exemplo abaixo:

        (video src="Ed Sheeran - Perfect (Official Music Video).mp4" controls>/video
